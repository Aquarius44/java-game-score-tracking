# 🎮 Java Game Score Tracking Application

## 📌 Project Overview
This is a comprehensive game score tracking and social interaction platform built with Java. Players can manage their scores while interacting with other users. It includes both game-related and community features.

## 🌟 Features
- 🔐 User Registration and Login
- 🔁 Forgot Password functionality
- 🧮 Add and View Game Scores
- 🧍‍♂️ View User Profiles
- 🤝 Add Friends and Chat
- 🚫 Block and Unblock Users

## 🔧 Technologies Used
- Java (Object-Oriented Programming)

# 🎮 Java Oyun Skor Takip Uygulaması

## 📌 Proje Hakkında
Bu proje, Java kullanılarak geliştirilmiş kapsamlı bir oyun skor takibi ve sosyal etkileşim platformudur. Kullanıcılar skorlarını takip edebilir ve diğer kullanıcılarla etkileşim kurabilir.

## 🌟 Özellikler
- 🔐 Kullanıcı Kayıt ve Giriş
- 🔁 Şifremi Unuttum özelliği
- 🧮 Skor Ekleme ve Genel Skorları Görme
- 🧍‍♂️ Profil Görüntüleme
- 🤝 Arkadaş Ekleme ve Mesajlaşma
- 🚫 Kişi Engelleme ve Engel Kaldırma

## 🔧 Kullanılan Teknolojiler
- Java (Nesne Yönelimli Programlama)
